# Python 쉬운 설명 (app)

이 문서는 초등학생도 따라할 수 있도록, 폴더 안 Python 파일을 아주 쉽게 설명해요.

## 먼저 기억해요
- `python 파일이름.py` 로 실행해요.
- 에러가 나면 메시지를 읽고 천천히 한 줄씩 확인해요.
- `run()` 함수가 있으면 그 함수가 핵심 실습이에요.

### main.py
- 이 파일은: 이 파일은 Python 실습 예제를 담고 있어요.
