# Python 쉬운 설명 (chapter98)

## 먼저 기억해요
- `python 파일이름.py` 로 실행해요.
- 에러가 나면 메시지를 읽고 천천히 한 줄씩 확인해요.
- `run()` 함수가 있으면 그 함수가 핵심 실습이에요.

### practice.py
- 이 파일은: 최종 미니 프로젝트 실습 파일
