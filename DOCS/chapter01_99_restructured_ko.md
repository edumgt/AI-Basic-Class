# AI/ML 응용 기초 재구성안 (Chapter01~Chapter99)

대상: 수학/통계학 지식이 없는 초등학생도 따라올 수 있는 입문자  
형식: 각 챕터 `10분 기본 지식 + 30분 Python 실습`

---

## 운영 원칙

- 수식 대신 말과 그림(그래프) 중심
- "정답 맞히기"보다 "패턴 찾기" 중심
- 매 챕터 산출물: 콘솔 출력 1개 + 그래프/표 1개
- 실습 난이도: `print -> list/dict -> pandas -> sklearn -> mini project`

---

## 기존 레포와 연결

- 기존 `chapter01~30`은 그대로 유지
- 아래 99챕터 로드맵은 기존 30챕터를 더 잘게 쪼개고, 초급자 친화 순서로 재배치한 확장안
- 우선 적용 순서:
1. Chapter01~33: 파이썬/데이터/ML 감각 만들기
2. Chapter34~66: 모델링/평가/개선 루프
3. Chapter67~99: 딥러닝 감각, 응용, 프로젝트
